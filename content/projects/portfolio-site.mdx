## Overview

This portfolio site is built with the Next.js App Router, Tailwind CSS v4, and a custom MDX pipeline.

## Key decisions

### Server Components by default

MDX compilation happens entirely on the server using `next-mdx-remote/rsc`. No MDX JS is shipped to the client.
